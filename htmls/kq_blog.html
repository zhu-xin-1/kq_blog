Django的使用

Django的安装等可去菜鸟教程学习
安装完成后pycharm中打开终端 键入：django-admin.py startproject kq_blog 即可完成一个项目的创立

Django采用MTV框架，即模型（models)、视图(views)以及模板（Template）

其中模型可简单理解为 对数据库操作的抽象化（使用ORM来映射数据库），根据url传来的参数在视图中处理后调用对应的模型
其中视图是为了完成业务逻辑，包括url连接，调用模型中的数据，以及直接返回HttpResponse 或者使用render，传入处理后的参数并调用模板返回给用户
其中模板就是返回给用户的HTML网页

一般的建立的步骤为
创建模型（比如用于存储用户账户密码的数据库表）
终端命令：django-admin.py startapp normal_user
这项目中会出现normal_user文件夹，其中models中就是创建我们需要的数据库的文件
在models中键入如下内容：
{% load static %}
相当于在数据库中建立了一个名normal_user的表，现在表中仅仅只有账户密码且都是明文的

python3 manage.py migrate # 创建表结构
python3 manage.py makemigrations normal_user # 让 Django 知道我们在我们的模型有一些变更
python3 manage.py migrate normal_user # 创建表结构
其中python3 可用py代替
